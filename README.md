# AI-Powered-Booking-Application
I created AI Powered Skill Workers Booking Application. Customer can enter keywords according to their needs and they can upload a image of issue as well. The System's algorithm can search and provide better client (skill Worker) by matching customer's location. System's AI algorithm can provide a better solution to fix it.
